# UI/UX Pro Max (UUPM)

## What It Is

Paid Claude Code skill with a curated design database — 57 UI styles, 95 color palettes, 56 font pairings, 29 landing page patterns. Generates production-ready code from design briefs. https://uupm.cc/

## Problem It Solves

Developers who can build anything but struggle to make it look good. Provides pre-vetted design decisions so you skip the research phase.

## Key Observations

- $5-39/mo subscription, 30–1500 requests/mo depending on tier
- Framework-aware: React, Next.js, Vue, Svelte, SwiftUI, etc.
- Includes accessibility checks, dark mode, conversion-optimized patterns
- 11k GitHub stars — decent adoption
- Essentially a curated knowledge base delivered as a Claude Code skill

## Verdict

**Skip.** The `frontend-design` skill already in our setup does the same job for free. Claude natively knows design patterns, color theory, and Tailwind UI generation. The marginal value of a curated database doesn't justify the subscription.

## Notes

- Revisit candidate if `frontend-design` skill proves insufficient for a specific project
